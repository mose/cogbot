# Cogbot

Cogbot is an irc bot written in ruby based on [Cinch bot framework](https://github.com/cinchrb/cinch).

## Installation

    $ gem install cogbot

## Usage

Edit the file config/cogbot.yml
launch cogbot:

    cogbot start
