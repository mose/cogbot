# Cogbot

Cogbot is an irc bot written in ruby based on Cinch bot framework.

## Installation

Add this line to your application's Gemfile:

    gem 'cogbot'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cogbot

## Usage

Edit the file config/cogbot.yml
launch cogbot
